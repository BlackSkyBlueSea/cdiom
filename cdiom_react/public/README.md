# 静态资源文件夹

此文件夹用于存放静态资源文件，可以直接通过路径访问。

## Logo 文件

请将 logo 图片文件放在此文件夹中，命名为：
- `logo.png` (推荐，如果是PNG格式)
- `logo.svg` (如果是SVG格式)
- `cdiom-logo.png` (也可以使用此名称)

### 使用方式

在代码中通过以下方式访问：
```jsx
<img src="/logo.png" alt="CDIOM" />
```

### Logo 要求

- **收起状态**：建议尺寸 32x32 像素（或等比例）
- **展开状态**：建议尺寸 32x32 像素（或等比例）
- **格式**：PNG（支持透明背景）或 SVG（矢量图，推荐）
- **文件大小**：建议小于 100KB

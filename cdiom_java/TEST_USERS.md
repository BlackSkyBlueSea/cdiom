## 使用说明

1. 使用超级管理员账号登录系统
2. 进入"用户管理"页面
3. 点击"新增用户"按钮
4. 按照上述信息填写表单：
   - 用户名：使用上表中的用户名
   - 手机号：使用上表中的手机号
   - 密码：使用上表中的密码
   - 角色：选择对应的角色
5. 提交后即可使用该账号登录测试

## 密码规则

所有测试密码都遵循以下规则：
- 至少6个字符
- 包含大小写字母
- 包含数字
- 包含特殊字符（@）

## 注意事项

- 用户名和手机号必须唯一
- 如果提示"用户名已存在"或"手机号已存在"，请修改用户名或手机号
- 密码在创建后会自动使用BCrypt加密存储
- 创建的用户默认状态为"正常"，可以直接登录使用
